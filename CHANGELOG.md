# Changelog

這份文件用來簡單記錄每個版本做了什麼。格式建議維持：

```text
## v版本號 - YYYY-MM-DD

- 變更 1
- 變更 2
- 修正 1
```

## v1.4.2 - 2026-07-16

- 新增歌曲標籤 SQL 與管理頁，預設上層分類為類型、語言、其他。
- 歌曲編輯改為獨立編輯頁，支援修改歌曲名稱與勾選歌曲標籤。
- 歌曲清單簡化為歌手名稱、歌曲名稱、歌曲長度與編輯按鈕。
- 歌曲編輯頁顯示原始 YouTube 標題、檔案位置、歌曲長度、上傳日期與狀態。
- 修正歌曲清單編輯按鈕被下方文字蓋住的版面問題。
- 修正歌手清單頭像載入，優先從本機 assets 讀取。
- 下載歌曲區搜尋改為按下搜尋按鈕或 Enter 後才套用，避免輸入時列表閃爍。

## v1.4.1 - 2026-07-15

- 調整歌手管理 UI：歌手清單改為第一個分頁，新增歌手改為第二個分頁。
- 歌手清單改為三欄固定大小卡片，僅顯示歌手名稱與 artist_id。
- 新增標籤管理 UI，支援上層與下層標籤新增、停用、恢復與刪除。
- 標籤管理新增排序功能，上層與下層標籤可用上下按鈕調整順序。
- 上層標籤選擇改為名稱前方的勾選框。
- 歌手編輯頁支援多選歌手標籤。
- 歌曲管理與歌手管理圖片改用持久化 assets 目錄，並支援從舊 cache 補回圖片。
- 歌曲管理頁優先同步讀取本機縮圖，減少顯示 No Image 的情況。
- 下載歌曲時會保存並更新歌曲長度與上傳日期，供後續檢視與篩選使用。
- 全站字體與主要按鈕尺寸調整，改善高解析度與全螢幕使用體驗。
